
# Fragrance App

A prototype version of my own fragrance management app



## Features

#### Planned:
- Huge fragrance database
- Adding new perfumes
- Making own fragrance collections
- Checking basic perfume info like saturation, accords, notes, etc.


#### Ideas:
- Checking the best prices in legitimate perfume stores
- Recommendations
